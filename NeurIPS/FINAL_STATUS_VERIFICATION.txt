DATASET - ADVERSARIAL-2K
================================================================================

CONSTRUCTION:
✅ 2000 balanced samples across 5 cross-modal failure modes
✅ 400 samples per failure category
✅ Difficulty stratified (baseline: 50.7% vs. human: 73%)
✅ Each sample: visual edit + textual edit + compositional test query
✅ All samples manually verified by annotators

FAILURE CATEGORIES:
1. Polysemy (400): Same token, multiple semantic meanings
   - Example: "Jaguar" (animal) vs. "Jaguar" (brand)
   
2. Near-Miss (400): Subtle differences causing confusion
   - Example: "apple" (fruit) vs. "Apple" (logo)
   
3. Conflict (400): Explicit image-text contradictions
   - Example: Image "red car" but text says "blue car"
   
4. Multi-hop (400): Multi-step cross-modal reasoning
   - Example: (A→B edit) then query: "Is image consistent with B?"
   
5. Hard Visual (400): Abstract concept visual grounding
   - Example: Concepts like "justice", "trust" with symbolic imagery

STATISTICS:
- Total samples: 2000
- Category balance: Perfect (400 each)
- Human baseline accuracy: 73%
- Model baseline accuracy: 50.7%
- Difficulty delta: 22.3 percentage points
- Dataset provides strong adversarial stress test
